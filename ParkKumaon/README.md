# ParkKumaon

## Problem

Tourists visiting destinations across Kumaon often struggle to find reliable parking, leading to wasted time, traffic congestion, and a poor travel experience.

## Solution

ParkKumaon is a two-sided marketplace that helps users discover, compare, and book parking spaces near popular tourist destinations in Uttarakhand.
