# M-Live

## Creator & Live Commerce Integration

### Problem Statement
Users discover products through creators but face friction between content consumption and purchase.

### Solution
M-Live integrates creator-led live commerce with seamless product discovery and checkout.

### Target Users
- Content Creators
- D2C Brands
- Online Shoppers

### Key Features
- Live Shopping Sessions
- Creator Product Tagging
- Interactive Engagement
- One-Click Purchase Flow

### Documents
- Product Requirement Document (PRD)
- Launch Strategy

### Skills Demonstrated
- Product Strategy
- User Research
- PRD Writing
- GTM Planning
- Marketplace Thinking
